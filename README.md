# TaskForge SaaS - Task Management Backend

A production-ready SaaS backend built with FastAPI, PostgreSQL, Docker, and Kubernetes.

## 🚀 Features
- User Authentication (JWT-based)
- Secure Password Hashing (bcrypt)
- RESTful API architecture
- PostgreSQL Database with SQLAlchemy ORM
- Dockerized microservices setup
- Kubernetes deployment (Minikube)

## 🧱 Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker & Docker Compose
- Kubernetes (Minikube)
- JWT Authentication

## 📦 Architecture
- API Service (FastAPI)
- Database Service (PostgreSQL)
- Containerized using Docker
- Orchestrated using Kubernetes
